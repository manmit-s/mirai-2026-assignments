# MirAI School of Technology (Summer 2026)

This repository serves as a centralized hub for all the assignments, project builds, and practical implementations completed during the **Summer Internship 2026 (AI Builder Track)** at MirAI School of Technology.

## 📁 Architecture
The repository is structured modularly. Each session's task is contained within its own dedicated directory:

```text
mirai-ai-builder-2026/
├── assignment-1/      # Task 1 source files
├── assignment-2/      # Task 2 source files
└── ...
```

## 🛠️ Environment Setup & Installation
To run any of the application modules locally, follow these steps:

1. **Clone the Repository:**

Bash

```
git clone [https://github.com/YOUR_GITHUB_USERNAME/mirai-ai-builder-2026.git](https://github.com/YOUR_GITHUB_USERNAME/mirai-ai-builder-2026.git)
cd mirai-ai-builder-2026
```
2. **Install Core Frameworks:**

Bash

```
pip install streamlit
```
3. **Execute a Specific Assignment:**
Navigate into the respective assignment folder or specify its path directly. For example, to run the first assignment:

Bash

```
streamlit run assignment-1/app.py
```
